============= Запуск =============

--------- через launcher.py ---------
P - установить порт для запуска сервера/клиента
S - запуск сервера (порт из конфига переопределен)
C - запуск клиента
Q - закрыть все процессы и выйти
(не рекомендуется, т.к. создает бд и конфиги в корне проекта)

--------- через скрипты ---------
Параметры запуска server/start_server.py:
[-p port] - порт сервера (default=7777)
[-a addr] - адрес хоста (default='')
если не указывать параметры будут использоваться параметры из конфига(server.cfg) или default

Параметры запуска client/start_client.py:
[addr] - адрес сервера (default='localhost')
[port] - порт сервера (default=7777)
параметры можно изменить в окне login

Прим.:
[] - необязательный параеметр

============= Описание функций =============
--------- Сервер ---------
Config - открывает окно конфигураций (для применения параметров нужен рестарт + запкуск через start_server.py)
Refresh - вкл/выкл обновление списков (обновление раз в 10 сек)

--------- Клиент ---------
окно Login:
- Username / Password:
работает и как регистрация и как авторизация, т.е.
если Username не зарегистирован, то создается юзер
если зарегистирован и введен неверный пароль, то будет выведено сообщение "Unauthorized"

основное окно:
- список Users:
* кнопка Add добавляет юзера в список контактов
* двойное нажатие загружает чат с юзером
- список Contacts:
* кнопка Rem удаляет юзера из список контактов
* двойное нажатие загружает чат с юзером

прим: загрузка чата происходит только если юзер "в сети",
это сделано из-за комбированного метода шифрования сообщений

============= Обновление ДЗ урок 1 =============
--------- Клиент ---------
- поле аватар (если аватара нет, то просто квадрат):
* двойное нажатие открывает окно загруки аватарки + установки фильтра
- поле чат:
* можно форматировать текст:
**<тескт>** - bold
##<тескт>## - italic
__<тескт>__ - underline
